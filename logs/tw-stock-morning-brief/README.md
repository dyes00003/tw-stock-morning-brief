# Taiwan Stock Morning Brief Logs

This directory stores one log file per automation run.

Expected filename pattern:

- `YYYY-MM-DD_HH-mm-ss.md`

Each log should record:

- run timestamp
- report date and price date
- theme rankings
- stock ranking changes versus the previous run
- key cross-border upstream and downstream supply-chain findings
- unverifiable items
- source links used in the run
- files updated by the automation
